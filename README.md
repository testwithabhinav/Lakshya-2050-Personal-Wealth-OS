# 🚀 Lakshya 2050 - Personal Wealth Operating System (PWOS)

> **A personal finance platform built using Excel, Power BI, Git and AI.**

---

## 📖 About

Lakshya 2050 is a long-term financial planning platform designed to help individuals and families make informed financial decisions.

The project combines financial planning with software engineering principles to create a maintainable and scalable Personal Wealth Operating System.

The initial release focuses on building a working MVP that manages income, expenses, loans, investments, insurance, assets, and financial goals.

---

# 🎯 Vision

To create a single platform capable of answering questions like:

- Can I afford to buy a car?
- Should I prepay my loan?
- Am I on track for retirement?
- What is my current net worth?
- How healthy are my finances?
